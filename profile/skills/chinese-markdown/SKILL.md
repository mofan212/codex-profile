---
name: chinese-markdown
description: 处理中文内容 Markdown / `.md` 文档的语法习惯和中文排版。当用户要求新增、修改、格式化或审查以中文为主要内容，或包含中文段落、中文列表、中文表格的 Markdown / `.md` 文档（例如需求文档、博客文章、检索文档、SKILL.md 等文档）时，使用该 Skill。不适用于纯英文 Markdown、业务事实改写、需求设计或非排版类内容润色。
---

# 1. 执行流程

| step | action | output |
| --- | --- | --- |
| 1 | 确认目标是中文 Markdown / `.md` 文档或中文 Markdown 片段 | 判定是否使用本 Skill |
| 2 | 判断用户是要求落地修改、只审查，还是限定只检查某一类问题 | 确定 `format_document`、`skill_markdown`、`markdown_limited_check`、`copywriting_limited_check`、`review_only` 或 `non_markdown` |
| 3 | 用户未限定只检查 Markdown 结构问题时，读取 [chinese-copywriting.md](references/chinese-copywriting.md) | 确定本次必须应用的中文排版规则 |
| 4 | 读取目标文档已有标题、列表、表格和行内语法习惯；如果只给片段，则只看片段内部一致性 | 确定本次遵循的局部格式基线 |
| 5 | 按用户限定范围处理；未限定时同时处理行内语法、标题规则、尾部标点和中文排版 | 产出修改内容或问题清单 |
| 6 | 按输出要求和最终检查清单汇报处理范围、修正类型、跳过项、语义变化和校验结果 | 给出可追溯的交付说明 |

# 2. 输入路由与边界

| input_type | match_signal | action | skip_when |
| --- | --- | --- | --- |
| `format_document` | 用户要求新增、修改、格式化、排版或修正中文 Markdown | 先读取 [chinese-copywriting.md](references/chinese-copywriting.md)，再同时检查 Markdown 语法习惯和中文排版 | 用户明确限定只检查某一类问题 |
| `skill_markdown` | 目标文件是 `SKILL.md` 或 Skill 文档，且任务涉及中文 Markdown 写作、格式、表格、列表、链接或排版 | 只应用 Markdown 和中文排版规则；Skill 任务设计、触发条件、工作流和资源规划由对应 Skill 创建、优化或维护流程负责 | 不接管 Skill 设计，不改业务语义 |
| `markdown_limited_check` | 用户说「只检查标题」「只看表格」「只检查链接」「不要改正文」等 Markdown 结构限定 | 只处理用户指定的 Markdown 结构范围，并说明未处理的问题类型，例如标题层级、标题编号、链接语法、表格列数、列表或表格尾部标点 | 不扩大到正文排版、措辞或其他 Markdown 规则 |
| `copywriting_limited_check` | 用户说「只检查中英空格」「只看中文标点」「检查全角半角」「统一专有名词大小写」等中文排版限定 | 先读取 [chinese-copywriting.md](references/chinese-copywriting.md)，再只处理用户指定的中文排版项 | 不扩大到 Markdown 结构、业务事实或其他未指定排版项 |
| `review_only` | 用户要求审查、指出问题或给修正方向，但没有要求落地修改 | 未限定只检查 Markdown 结构问题时，读取 [chinese-copywriting.md](references/chinese-copywriting.md)；输出问题列表、依据和修正方向，不改文件；问题列表必须覆盖标题层级、标题编号、行内语法空格、尾部标点和中文排版 | 用户随后明确要求修改 |
| `non_markdown` | 目标不是 Markdown / `.md` 文档，或任务目标主要是需求设计、事实改写、业务润色 | 不使用本 Skill，按普通任务处理；如果对象是 Markdown 需求文档但用户只要求格式、排版或审查 Markdown 语法，则只处理格式问题 | 用户确认转为中文 Markdown 排版任务 |

# 3. 失败处理与检查点

| trigger | first_action | fallback | stop |
| --- | --- | --- | --- |
| 用户只粘贴片段，无法判断完整文档既有格式 | 只按片段内部一致性处理 | 在反馈中说明未读取完整文档，不能保证全局标题编号连续 | 不推断完整文档结构 |
| 用户限定只检查某一类问题，但文本中存在限定范围外的可直接定位问题 | 只处理限定范围 | 在最终说明中简短提示未处理的问题类型 | 不主动扩大修改范围 |
| 需要应用中文排版但无法读取 [chinese-copywriting.md](references/chinese-copywriting.md) | 🛑 STOP：说明读取失败，不能继续中文排版检查 | 用户提供排版规则内容，或明确确认本轮降级为只处理本文件已载入的 Markdown 语法习惯 | 未获得确认前不凭记忆替代 reference，也不声称已完成中文排版检查 |
| 文档已有规则与本 Skill 不同 | 优先保持目标文档既有规则 | 如果既有规则导致表格列数不一致、链接语法破损、代码围栏未闭合或标题编号断裂，说明冲突并给出选择 | 不擅自统一整篇风格 |
| 修改可能改变业务含义、术语、事实或结论 | 🔴 CHECKPOINT：只做格式层修改，并向用户确认是否允许语义改写 | 用户确认后再做内容润色或事实调整 | 未确认前不改业务事实 |
| 需要重排标题层级、拆分章节或合并大段内容 | 🛑 STOP：先说明结构影响范围和预计改动 | 用户确认后再执行结构调整 | 不静默重排整篇结构 |

# 4. Markdown 规则

- Markdown 行内代码、链接、强调等语法与中文内容相邻时，两侧留一个半角空格
- Markdown 行内语法与中文标点相邻时，不强制在标点前后加空格；行内代码后接顿号、逗号、句号、分号、冒号、问号或感叹号时，保持紧邻
- 同一文档中同一字段名、配置项或命令参数已经稳定使用行内代码时，表格中的同名内容可以统一为行内代码；不要因此改写字段名、配置项或命令参数本身
- 文章内容最多只使用 1 级标题和 2 级标题；2 级标题必须属于前一个 1 级标题
- 每个标题前都要指定序号，并按层级连续递增
- 不要只保留一个 1 级标题并把所有内容塞进 2 级标题
- 需要更细层级时，优先使用引用块或列表，不要继续新增 3 级或更深标题
- Markdown 无序列表的条目末尾不要添加句号、分号、冒号、逗号、顿号等收尾符号
- Markdown 有序列表的条目末尾可以不加收尾符号；如使用收尾符号，非最后一项仅可用分号，最后一项仅可用句号
- 列表项作为小标题，并引出紧随其后的表格、代码块、引用块或子列表时，可以保留冒号；这类冒号属于结构提示，不算普通尾部标点
- Markdown 表格内文字末尾也不要添加这些收尾符号
- 只有条目或单元格内容本身必须保留原文标点时，才保留原文
- 实际修改或审查行内语法、标题层级、尾部标点时，读取 [markdown-examples.md](references/markdown-examples.md)；用户限定检查项且不涉及这三类规则时跳过

# 5. 输出要求

| item | requirement |
| --- | --- |
| 处理范围 | 说明处理了全文、指定章节、粘贴片段还是用户限定项 |
| 修正类型 | 概括修正了哪些类型，例如行内空格、标题编号、列表标点、表格尾部标点 |
| 跳过项 | 用户限定范围时，说明哪些可直接定位问题类型未处理 |
| 语义变化 | 默认说明无语义变化；如果改动涉及语义，说明用户确认结果 |
| 校验结果 | 说明是否完成最终检查清单 |

# 6. 禁止行为

| anti_pattern | forbidden | replacement |
| --- | --- | --- |
| 借排版改语义 | 不要把格式化任务扩大成业务事实、术语、结论或观点改写 | 只改 Markdown 语法和中文排版；语义改写需用户确认 |
| 忽略限定范围 | 不要在用户说「只检查标题」时顺手修改正文、链接、表格或措辞 | 只处理限定范围，并提示未处理项 |
| 强行统一全文风格 | 不要忽略目标文档已有标题、列表、表格和行内语法习惯 | 先遵循目标文档局部一致性 |
| 机械新增深层标题 | 不要为了表达层级新增 3 级或更深标题 | 使用引用块、列表或拆分为 1 级和 2 级标题 |
| 过度改写代码标识 | 不要为了排版改写字段名、配置项、命令参数或链接目标 | 只修正空格、标点和同文档内既有格式一致性 |

# 7. 最终检查

交付前检查：

- Markdown 行内语法两侧与中文内容留有一个半角空格
- Markdown 行内语法与中文标点紧邻时没有误判为缺少空格
- 未限定只检查 Markdown 结构问题时，已按 [chinese-copywriting.md](references/chinese-copywriting.md) 检查中文与英文、数字、单位之间的空格
- 已检查中文标点、全角半角、直角引号、重复标点和专有名词大小写
- 标题只使用 1 级和 2 级；审查模式也必须指出 3 级或更深标题
- 标题编号连续且层级正确；审查模式也必须指出缺少编号、编号断裂或层级不匹配
- 无序列表条目末尾没有多余标点；有序列表未使用收尾符号，或仅在非最后一项使用分号、最后一项使用句号
- 引出后续表格、代码块、引用块或子列表的列表项可以保留冒号
- 表格内文字末尾没有多余标点
- 输出说明覆盖处理范围、修正类型、语义变化和校验结果；用户限定范围时补充说明跳过项
